# 目录
 家族树
 * bootstrap框架包含样式以及jquery
 * img里面放图片
 * main里面为进入页面代码
 * tree里面为生成家族树代码（文本框）
 
 # 使用方法
所有文件下载到本地，压缩包解压，打开main，进入生成家族树的页面，在文本框中输入家族树文本形式，点击“生成”，即刻生成家族树
 （建议使用eclipse，因为图片皆保存在相对路径）
